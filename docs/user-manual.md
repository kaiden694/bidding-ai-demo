# 用户手册

> 智能招投标与合同合规 AI 工作台

## 1. 系统概述

智能招投标与合同合规 AI 工作台是一个基于大模型与 RAG 的招投标全流程辅助系统，覆盖：
- 文档解析与参数偏离比对
- 合同风险扫描
- 知识库管理（历史标书/经验/范本）
- 资质台账与到期预警
- AI 助手
- 协作评论与待办

### 1.1 角色说明

| 角色 | 职责 | 主要权限 |
|------|------|----------|
| admin | 系统管理 | 所有权限 |
| presales | 售前/标书 | 项目/文档/比对/知识库/资质/标书起草 |
| legal | 法务/合同 | 合同审查/风险扫描/反馈 |
| procurement | 采购 | 资质/产品/合同 |
| pm | 项目经理 | 项目管理/全流程跟踪 |
| compliance | 合规 | 合同审查/审计 |

## 2. 操作指南

### 2.1 登录
1. 访问系统地址（如 http://localhost:5173）
2. 输入用户名和密码
3. 默认管理员：admin / admin123（**首次登录后请立即修改密码**）

### 2.2 项目管理
1. **创建项目**：项目管理 → 新建 → 填写名称/代码/甲方
2. **状态流转**：项目详情 → 选择下一状态 → 填写流转原因
   - 状态机：筹备→立项→文件解析→方案规划→标书起草→内审→投递→评标→中标/落标→归档
3. **AI 推荐下一状态**：点击"AI 推荐"获取智能建议

### 2.3 文档解析
1. 上传招标文件/规格书/合同（PDF/Word）
2. 系统自动解析为结构化切块
3. 查看解析结果（按章节/页码定位）

### 2.4 参数偏离比对
1. 选择项目 → 新建比对任务
2. 选择规格书 + 标书/投标文件
3. 系统自动比对参数偏离
4. 查看比对结果（偏离项 + 证据链）

### 2.5 合同风险扫描
1. 选择合同 → 启动风险审查
2. AI 扫描合同条款（付款/违约/知识产权/保密等）
3. 查看风险报告（风险等级 + 条款定位 + 修正建议）

### 2.6 知识库管理
1. **历史知识库**：上传历史标书/合同/经验 → 自动切块向量化
2. **通用知识库**：行业通用规则/术语词典
3. **批量导入**：ZIP 文件批量上传
4. **索引重建**：更新 Embedding 模型后重建索引

### 2.7 资质管理
1. 创建资质台账记录
2. 上传证书 PDF
3. 触发 OCR + LLM 字段提取（自动提取名称/编号/有效期/范围）
4. 查看即将过期资质（预警视图）
5. Celery 每日 09:00 自动扫描到期资质

### 2.8 标书起草
1. 选择模板（技术方案/商务条款/资质清单/项目案例）
2. 填充模板变量
3. AI 生成章节草稿（基于历史标书 + 知识库 RAG）
4. 编辑草稿 → 保存

### 2.9 待办与通知
1. **待办看板**：查看分配给自己的待办任务
2. **通知中心**：站内信（状态变更/@提及/到期提醒）
3. 状态变更自动生成待办

### 2.10 协作评论
1. 在合同/文档/比对页面发表评论
2. @提及同事（自动通知）
3. 嵌套回复

### 2.11 AI 助手
1. 与 AI 助手对话，查询业务问题
2. AI 基于知识库 RAG 回答（带证据引用）

## 4. 常见问题

### Q1: 忘记密码怎么办？
A: 联系管理员重置密码。

### Q2: AI 功能返回错误怎么办？
A: 可能 LLM 服务暂时不可用，稍后重试。如持续错误，联系管理员检查 LLM 提供商状态。

### Q3: 文档解析失败怎么办？
A: 检查文件格式（支持 PDF/Word），如扫描件可能需要 OCR。联系管理员查看日志。

### Q4: 比对结果不准确怎么办？
A: 使用"反馈"功能提交修正，系统会学习并提升准确率。

### Q5: 如何获取帮助？
A: 联系系统管理员或查阅本文档。
